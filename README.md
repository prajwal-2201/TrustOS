## 🕵️‍♂️ The Mission
The internet is no longer a safe space. Deepfakes, forged payment receipts, and malicious QR codes are the weapons of modern digital warfare. **Trust OS** is the shield. Built with a combination of **Machine Learning Neural Networks**, **Error Level Analysis (ELA)**, and **OCR Forensic Engines**, it identifies truth in milliseconds.

> *“Trust is a luxury. Verification is a necessity.”*

---

## 🔥 Lethal Features

### 🔍 Forensic Image Analysis (ELA)
Our engine doesn't just look at pixels; it looks at the **digital history** of the image. By analyzing JPEG compression differences, Trust OS exposes exactly where an image has been manipulated—whether it's a forged amount on a bank receipt or a face swap in a deepfake.

### 🧠 Neural Scam Detection
Powered by a custom-trained **Scikit-Learn Logistic Regression model**, Trust OS analyzes the linguistic patterns of a message. It predicts fraud with clinical precision, identifying social engineering tactics before they land.

### 🕸️ Phishing & QR Decryption
Hidden URLs in QR codes are the preferred entry point for cyberattacks. Trust OS automatically extracts, decrypts, and evaluates these links against real-time threat intelligence.

### ⚡ Professional Forensic Summary
Get a detailed breakdown of every threat:
- **Risk Score (0-100)**: Instant situational awareness.
- **ML Probability**: AI-driven certainty metrics.
- **Forgery Traces**: Visual confirmation of pixel manipulation.
- **Copyable Forensic Reports**: One-click reports for law enforcement or security audits.

## 🚀 Deployment Instructions

### 1️⃣ Arm the System
Clone the terminal and install the dependencies:
```bash
git clone https://github.com/prajwal-2201/trustos.git
cd trustos/backend
pip install -r requirements.txt
```

### 2️⃣ Operational Requirements
Ensure you have **Tesseract OCR** installed on your host system for text extraction.

### 3️⃣ Ignite the Engine
Launch the server to start the live dashboard:
```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```
✨ **Access the Hub**: Open `http://localhost:8000` to begin your first investigation.
